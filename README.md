### Medidas de Impureza

1. **Error de Clasificación (IGE)**:
   - Error de Clasificación: \( IGE = 1 - \text{max}(P(i|t)) \)

2. **Impureza de Gini (IGG)**:
   - Impureza de Gini: \( IGG = 1 - \sum (P(i|t)^2) \)

3. **Entropía (IGH)**:
   - Entropía: \( IGH = - \sum (P(i|t) \cdot \log_2(P(i|t))) \)
